Accounts & Profiles

Accounts and profile options

This page describes the current DoseGraph account types and the profile-generation limits attached to each one. Full account management tools are still in construction, but the account model is already defined.

Patient account

Up to 5 profiles

Intended for personal or single-user tracking. This keeps the free core experience accessible while limiting profile volume.

Clinician account

Up to 20 profiles

Intended for professional review workflows and multiple active patient cases while still remaining part of the free-access model.

Developer account

Unlimited profiles

Intended for engineering, testing, prototyping, and product-development workflows where profile limits would create friction.

Account generation options

  • Guest/local use currently behaves like the patient tier.
  • Signed-in standard accounts currently behave like the clinician tier.
  • Developer and admin accounts have unlimited profile generation.

Current behavior

  • Users can continue to graph and edit data even after reaching a profile limit.
  • Profile limits only block creation of new profiles.
  • Existing profiles can still be loaded, updated, or deleted.

Status

Full accounts and profiles management controls are still under construction. For now, sign-in, profile saving, loading, and graph persistence continue to live in the main DoseGraph workspaces.
