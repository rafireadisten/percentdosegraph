🚀 GitHub Setup Helper

Get your PercentDoseGraph codebase on GitHub in 3 easy steps

✅ Automated Setup (Easiest)

Use this command in your terminal to automatically create your GitHub repo and push your code:

cd /Users/readisten/Documents/percentdosegraph ./setup-github.sh

This will:

  • Check your GitHub login
  • Create the repository automatically
  • Push all your code
  • Show you the GitHub URL
Note: Requires GitHub CLI to be installed
Install GitHub CLI Use Manual Instead

📋 Step 1: Create Repository on GitHub

Go to GitHub and create a new repository:

Create Repository

Fill in these fields:

  • Repository name: percentdosegraph
  • Description: A lightweight clinical dosing visualization tool
  • Visibility: Public

Then click "Create repository"

📝 Step 2: Copy and Run Commands

Find your GitHub username in the URL you just created. For example:

https://github.com/YOUR_USERNAME/percentdosegraph

YOUR_USERNAME = your GitHub account name

Then run these commands in your terminal:

cd /Users/readisten/Documents/percentdosegraph git remote add origin https://github.com/YOUR_USERNAME/percentdosegraph.git git branch -M main git push -u origin main

🎉 Step 3: Verify on GitHub

Your code should now be visible at:

https://github.com/YOUR_USERNAME/percentdosegraph

⚠️ Important: Replace YOUR_USERNAME with your actual GitHub username!

🔐 Next: Configure Secrets

After your code is on GitHub, add your API secrets:

  1. Go to your repository Settings
  2. Click "Secrets and variables" → "Actions"
  3. Add these secrets:
    • JWT_SECRET (required)
    • GOOGLE_CLIENT_ID (optional)
    • GOOGLE_CLIENT_SECRET (optional)
Go to Secrets Settings

Need help? See GITHUB_HOW_TO.md in your project for detailed instructions
